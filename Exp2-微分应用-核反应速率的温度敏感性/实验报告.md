# 3-α 反应速率的温度敏感性指数计算实验报告
## 一、实验目的
1. 理解3-α反应速率对温度的依赖性
2. 掌握数值微分方法（有限差分）计算温度敏感性指数的原理和实现
3. 分析不同温度下反应速率的敏感性特征及其物理意义
## 二、实验原理
### 2.1 3-α反应速率
3-α反应是一个重要的核聚变过程，将三个氦-4原子核聚变成一个碳-12原子核：

$${}^4\mathrm{He} + {}^4\mathrm{He} + {}^4\mathrm{He} \rightarrow {}^{12}\mathrm{C}$$

其能量产生率表达式为：

$$q_{3\alpha} = 5.09\times 10^{11} \rho^2 Y^3 T_8^{-3} e^{-44.027/T_8} \mathrm{erg g^{-1}~s^{-1}}$$

### 2.2 温度敏感性指数
温度敏感性指数 $\nu$ 定义为反应速率对数对温度对数的导数：

$$\nu = \left . \frac{d\log q}{d\log T}\right | _{T_0} = \left ( \frac{T}{q} \frac{dq}{dT} \right ) _{T_0}$$

使用前向差分法近似计算导数：

$$\nu(T_0) \approx \frac{T_0}{q(T_0)} \frac{q(T_0 + h \cdot T_0) - q(T_0)}{h \cdot T_0}$$

## 三、代码实现
（此处粘贴您实现的代码，包括反应速率函数和温度敏感性指数计算函数）

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置字体为黑体
plt.rcParams['font.family'] = 'SimHei'

def q3a(T):
    """
    计算 3-alpha 反应速率中与温度相关的部分 q / (rho^2 Y^3)
    输入: T - 温度 (K)
    返回: 速率因子 (erg * cm^6 / (g^3 * s))
    """
    # 处理零温度和负温度的情况
    if T <= 0:
        return 0
    # 将温度转换为以 10^8 K 为单位
    T8 = T / 1e8
    return 5.09 * 1e11 * T8**(-3) * np.exp(-44.027 / T8)


def plot_rate(filename="rate_vs_temp.png"):
    """绘制速率因子随温度变化的 log-log 图"""
    # 生成温度数据点
    temperatures = np.logspace(7, 10, 100)
    # 计算对应的速率值
    rates = [q3a(T) for T in temperatures]
    # 绘制双对数图
    plt.loglog(temperatures, rates)
    plt.xlabel('温度 (K)')
    plt.ylabel('q_3α / (ρ²Y³) (erg * cm⁶ / (g³ * s))')
    plt.title('3-α 反应速率与温度关系')
    plt.grid(True)
    plt.savefig(filename)
    plt.show()


if __name__ == "__main__":
    # 计算并打印 nu 值
    print("   温度 T (K)    :   ν (敏感性指数)")
    print("--------------------------------------")

    temperatures_K = [1.0e8, 2.5e8, 5.0e8, 1.0e9, 2.5e9, 5.0e9]
    h = 1.0e-8  # 扰动因子

    for T0 in temperatures_K:
        q_T0 = q3a(T0)
        # 避免 q_T0 为零的情况
        if q_T0 == 0:
            nu = 0
        else:
            delta_T = h * T0
            q_T0_plus_delta_T = q3a(T0 + delta_T)
            # 使用前向差分计算导数
            dq_dT = (q_T0_plus_delta_T - q_T0) / delta_T
            # 计算敏感性指数 nu
            nu = (T0 / q_T0) * dq_dT
        print(f"{T0:16.2e} : {nu:16.2e}")

    # 调用绘图函数展示结果
    plot_rate()
```

### 4.3 结果分析
1. 温度敏感性随温度变化的趋势分析：
   
   - 在低温区域（~10⁸ K）的敏感性特征
   
   \(\nu \approx 41\)，绝对值极大，说明反应速率对温度极其敏感。这是因为指数项的导数 \(d(e^{-A/T_8})/dT\) 随 \(T_8\) 减小而急剧增大，主导了总导数。
   
   - 在高温区域（~10⁹ K）的敏感性特征
   
   \(\nu\) 逐渐减小并变为负值。此时指数项的温度依赖性减弱（导数趋近于零），幂律项 \(T_8^{-3}\) 的负导数逐渐主导，导致速率随温度升高而下降（负敏感性）。
   
   - 敏感性指数的变化趋势及其物理意义
   
   低温区域的高正敏感性：
反应速率公式 \(q_{3\alpha} = 5.09 \times 10^{11} \rho^2 Y^3 T_8^{-3} e^{-44.027/T_8}\) 中，指数项 \(e^{-44.027/T_8}\) 是主导因素（\(T_8 = T/10^8\)）。低温时，\(T_8 \ll 44.027\)，指数函数对温度变化极其敏感（导数与指数项本身成正比），导致反应速率随温度升高呈指数级增长，\(\nu\) 因此显著为正。这意味着在低温环境下（如恒星演化初期的氦燃烧阶段），温度的微小升高会引发反应速率的剧烈增加，可能成为驱动恒星核心能量爆发或稳定性调节的关键因素。

高温区域的低敏感性及负趋势：
随着温度升高，\(T_8\) 增大，指数项的指数绝对值减小（\(44.027/T_8\) 降低），指数函数的增长速率减缓，而幂律项 \(T_8^{-3}\) 的负贡献（随温度升高反应速率下降的趋势）逐渐显现。当温度足够高时，幂律项的导数（负贡献）超过指数项的导数（正贡献），导致 \(\nu\) 转为负值。此时，反应速率的增长主要受指数项控制（仍随温度升高而增加），但增长率随温度升高而降低（即二阶导数为负）。这一特性在高温恒星环境中具有重要意义，例如在碳燃烧阶段，负敏感性可抑制反应速率随温度的过度增长，帮助恒星维持热平衡，避免失控性核聚变。

趋势的物理本质：\(\nu\) 的变化反映了反应速率中指数项与幂律项的竞争关系：\(\nu = \frac{T}{q} \frac{dq}{dT} = \frac{T}{q} \left( q \cdot \left( \frac{-3}{T} + \frac{44.027}{T^2} \right) \right) = -3 + \frac{44.027}{T_8}\)
（推导：对 \(q_{3\alpha}\) 取自然对数后求导，再转换为以温度对数为变量的导数）
可见，\(\nu = 44.027/T_8 - 3\)，直接表明 \(\nu\) 随 \(T_8\) 升高呈线性下降，当 \(T_8 > 44.027/3 \approx 14.68\)（即 \(T > 1.468 \times 10^9 \, \text{K}\)）时，\(\nu\) 转为负值。这一数学关系与实验结果完全一致，揭示了敏感性指数变化的本质是指数激活能（\(44.027/T_8\)）与幂律衰减（\(-3\)）共同作用的结果。

3. 数值计算方法的精度分析：

   - 步长选择（h = 10⁻⁸）的合理性
   
   平衡了截断误差（与 h 成正比，前向差分误差为 \(O(h)\)）和舍入误差（随 h 减小而增大）。过小的 h 会因浮点数精度限制导致差分结果失真，过大的 h 会引入显著截断误差。

   - 可能的数值误差来源及其影响
   主要为前向差分的一阶精度和浮点数运算误差。高温区 \(q(T)\) 变化平缓，差分结果对 h 更敏感，可能导致 \(\nu\) 的符号波动（如从正转负）。

## 五、讨论与思考
1. 为什么在低温（T₀ = 10⁸ K）时 $\nu$值特别大？这对恒星演化有什么意义？

低温时 \(\nu\) 极大，意味着温度微小升高会导致反应速率急剧增加，这是恒星演化中 “氦闪” 现象的重要机制 —— 当恒星核心温度达到 \(\sim 10^8 \, \text{K}\) 时，3-α 反应速率爆发式增长，促使核心快速升温并合成碳元素。

2. 数值微分中步长的选择（h = 10⁻⁸）是如何平衡截断误差和舍入误差的？

前向差分的截断误差随 h 减小而降低，但计算机对极小 h 的浮点数表示存在精度限制（如双精度数约 \(10^{-16}\) 的绝对误差）。选择 \(h = 10^{-8}\) 确保 \(\Delta T\) 足够小以近似导数，同时避免舍入误差主导。

3. 如果使用中心差分法替代前向差分法，计算结果会有什么不同？精度会如何改变？

中心差分公式 \(\frac{dq}{dT} \approx \frac{q(T_0 + h/2) - q(T_0 - h/2)}{h}\) 具有 \(O(h^2)\) 精度，误差更小且对称性更好。相比前向差分，中心差分在相同 h 下能提供更精确的导数，尤其在函数梯度变化平缓的高温区效果更明显。

4. 反应速率对温度的强依赖性对恒星内部的温度调节机制有什么影响？

低温高敏感性形成正反馈：温度升高→反应速率剧增→能量释放增加→温度进一步升高，可能导致核心不稳定（如氦闪）；高温负敏感性形成负反馈：温度升高→反应速率下降→能量释放减少→温度回落，有助于稳定恒星内部能量平衡。

## 六、结论
（总结实验的主要发现和结论）

3-α 反应速率对温度的依赖由幂律项和指数项共同决定，低温时指数项主导，高温时幂律项影响增强。

温度敏感性指数 \(\nu\) 随温度升高显著下降，从低温区的强正敏感性（\(\nu \approx 41\)）逐渐转变为高温区的负敏感性（\(\nu < 0\)），反映了反应速率对温度响应机制的转变。

数值微分方法（前向差分）能有效计算 \(\nu\)，但需合理选择步长以平衡截断误差与舍入误差。中心差分法可进一步提高精度。

物理意义：低温高敏感性是恒星演化中氦燃烧爆发的关键因素，而高温负敏感性则可能参与恒星内部的温度稳定调节，体现了核聚变反应与恒星动力学的复杂耦合。
