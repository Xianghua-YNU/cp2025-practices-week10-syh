# 数值积分方法比较实验报告（Simpson 法则 vs 梯形法则）

## 一、实验目的

1. 理解并掌握 Simpson 法则与梯形法则的数值积分原理及实现方法。
2. 通过实验比较两种方法在积分精度和误差方面的表现。
3. 能够分析不同子区间数 $N$ 对积分结果的影响。

## 二、实验原理

### 2.1 Simpson 法则

简述 Simpson 法则的积分思想、公式推导及适用条件（$N$ 必须为偶数），并给出公式：
$$
\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]
$$

### 2.2 梯形法则

简述梯形法则的积分思想、公式推导，并给出公式：
$$
\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]
$$

```python
import numpy as np

# 待积分函数
def f(x):
    return x**4 - 2*x + 1

# 梯形法则积分函数
def trapezoidal(f, a, b, N):
    """
    梯形法数值积分
    :param f: 被积函数
    :param a: 积分下限
    :param b: 积分上限  
    :param N: 子区间数
    :return: 积分近似值
    """
    h = (b - a) / N
    x = np.linspace(a, b, N + 1)
    y = f(x)
    return (h / 2) * (y[0] + 2 * np.sum(y[1:N]) + y[N])

# Simpson法则积分函数
def simpson(f, a, b, N):
    """
    Simpson法数值积分
    :param f: 被积函数
    :param a: 积分下限
    :param b: 积分上限
    :param N: 子区间数（必须为偶数）
    :return: 积分近似值
    """
    if N % 2 != 0:
        raise ValueError("Simpson 法则要求 N 必须为偶数")
    h = (b - a) / N
    x = np.linspace(a, b, N + 1)
    y = f(x)
    return (h / 3) * (y[0] + 4 * np.sum(y[1:N:2]) + 2 * np.sum(y[2:N-1:2]) + y[N])

def main():
    a, b = 0, 2  # 积分区间
    exact_integral = 4.4  # 精确解

    for N in [100, 1000]:  # 不同子区间数
        # 调用积分函数并计算误差
        trapezoidal_result = trapezoidal(f, a, b, N)
        simpson_result = simpson(f, a, b, N)

        # 计算相对误差
        trapezoidal_error = abs(trapezoidal_result - exact_integral) / exact_integral
        simpson_error = abs(simpson_result - exact_integral) / exact_integral

        # 输出结果
        print(f"N = {N}")
        print(f"梯形法则结果: {trapezoidal_result:.8f}, 相对误差: {trapezoidal_error:.2e}")
        print(f"Simpson法则结果: {simpson_result:.8f}, 相对误差: {simpson_error:.2e}")
        print("-" * 40)

if __name__ == '__main__':
    main()
```

### 4.2 误差分析
- 分析两种方法在不同 $N$ 下的误差表现。
- 
  梯形法则：当 \(N=100\) 时，相对误差为 \(2.42 \times 10^{-4}\)；当 \(N=1000\) 时，相对误差降至 \(2.42 \times 10^{-6}\)。误差随 N 增大而显著减小，但下降幅度与 N 的平方成反比。

  Simpson 法则：当 \(N=100\) 时，相对误差为 \(9.70 \times 10^{-9}\)；当 \(N=1000\) 时，相对误差降至 \(9.70 \times 10^{-13}\)。误差随 N 增大急剧减小，下降幅度与 N 的四次方成反比。
  
- Simpson 法则与梯形法则误差的数量级差异及原因。
  
  数量级差异：相同 N 下，Simpson 法则的误差比梯形法则低约 4~8 个数量级。例如，\(N=100\) 时，梯形法则误差是 Simpson 法则的约 \(2.5 \times 10^4\) 倍；\(N=1000\) 时，这一差距扩大至 \(2.5 \times 10^6\) 倍。

  原因：

  梯形法则基于线性插值（直线近似），其局部截断误差为 \(O(h^3)\)，整体误差为 \(O(h^2)\)（\(h = \frac{b-a}{N}\)），即误差与子区间长度的平方成正比。

  Simpson 法则基于二次多项式插值（抛物线近似），其局部截断误差为 \(O(h^5)\)，整体误差为 \(O(h^4)\)，即误差与子区间长度的四次方成正比。
由于 \(h \propto \frac{1}{N}\)，梯形法则误差随 N 增大呈 二阶收敛，而 Simpson 法则呈 四阶收敛，后者收敛速度远快于前者。

- 讨论 $N$ 增大时误差的变化趋势。
- 
  两种方法的误差均随 N 增大而减小，符合数值积分方法的收敛性理论。

  梯形法则：误差按 \(O(1/N^2)\) 量级下降。例如，N 从 100 增至 1000（增大 10 倍），误差从 \(2.42 \times 10^{-4}\) 降至 \(2.42 \times 10^{-6}\)，接近 \(1/100\)（即 \(10^{-2}\)）的比例，符合二阶收敛特性。

  Simpson 法则：误差按 \(O(1/N^4)\) 量级下降。例如，N 增大 10 倍时，误差从 \(9.70 \times 10^{-9}\) 降至 \(9.70 \times 10^{-13}\)，接近 \(1/10000\)（即 \(10^{-4}\)）的比例，体现四阶收敛的高速性。

## 五、结论
- 总结两种方法的适用场景和优劣。

  Simpson 法则：
  
优势：高精度（四阶收敛），适合光滑函数，误差随区间数增大快速下降。

适用：科学计算、精密工程等高精度需求场景。

局限：要求区间数为偶数，实现稍复杂。


梯形法则：
优势：简单易用，无区间数奇偶限制，计算效率高。

适用：快速估算、初步分析、资源受限场景（如嵌入式设备）。

局限：精度低（二阶收敛），误差下降慢。

选择建议：精度优先选 Simpson，效率或灵活性优先选梯形。

- 结合本实验结果，说明实际工程中如何选择数值积分方法。
  
  高精度优先：若问题对积分精度要求较高（如物理建模、精密计算），且 N 可设为偶数，优先选择 Simpson 法则，其误差随 N 增大快速减小，能以较少的计算量达到更高精度。

  简单快速优先：若问题只需粗略估算（如数据预处理、初步验证），或 N 需设为奇数（如硬件限制导致的离散化步长限制），梯形法则更适用，其简单性和灵活性在工程中具有实用价值。
